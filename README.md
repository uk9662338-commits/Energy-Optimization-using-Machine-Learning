# Energy Optimization using Machine Learning

## Overview
This project is a Machine Learning based Energy Optimization System developed using Django. It predicts energy consumption, detects anomalies, identifies peak usage hours, and provides smart energy optimization suggestions.

## Features
- Energy Consumption Prediction
- Peak Hour Detection
- Anomaly Detection
- Data Visualization
- Smart Optimization Suggestions
- User-Friendly Web Interface

## Technologies Used
- Python
- Django
- Machine Learning
- HTML/CSS
- JavaScript
- SQLite

## Future Improvements
- Real-time IoT Integration
- Advanced ML Models
- Cloud Deployment
- User Authentication System
